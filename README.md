# JWT Decoder

Paste any JWT token and instantly inspect its header, payload, and expiry status — no server, no tracking.

## Features

- Decodes header and payload without verification
- Expiry check with human-readable time remaining
- Formatted JSON output with syntax highlighting
- Algorithm badge (HS256, RS256, etc.)
- Copy decoded sections to clipboard
- Runs 100% in the browser — your tokens never leave your device
